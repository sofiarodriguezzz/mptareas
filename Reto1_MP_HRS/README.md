# Clasificación de Pingüinos - Modelado Predictivo

Este proyecto compara dos enfoques de clasificación:

1. Clasificador humano: reglas diseñadas manualmente basadas en la observación de las variables.
2. Modelo de Machine Learning: un árbol de decisión entrenado con los datos.

Dataset utilizado: Penguins Dataset.

Variables utilizadas:
- bill_length_mm
- bill_depth_mm
- flipper_length_mm
- body_mass_g

Modelo utilizado:
- Decision Tree Classifier

## Instalación

Instalar dependencias:

pip install -r requirements.txt

## Ejecutar el proyecto

python main.py

## Objetivo

Comparar el desempeño de reglas heurísticas humanas contra un modelo de aprendizaje automático.
